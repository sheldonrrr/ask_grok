## GitHub Release Notes (v1.5.0)

### New Features
- **Set Default AI After Adding** - After successfully adding an AI provider (single-AI mode), you can choose Yes/No to set it as the default AI; the Config dialog refreshes immediately

### Improvements
- **Clearer Kimi Platform Labels** - Kimi platform names follow the UI language (e.g. International / China Mainland, 国际版 / 中国大陆版) instead of staying stuck in a previous language or the ambiguous “domestic” wording
