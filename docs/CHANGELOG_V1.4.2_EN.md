## GitHub Release Notes (v1.4.2)

### New Features
- **AI Search** - Search your entire library using natural language. Open via menu, Ctrl+Shift+L shortcut, or simply open Ask dialog without selecting any books. Ask questions like "Do you have any books about Python?" and AI will recommend relevant books from your library.
- **Direct AI Search Access** - AI Search menu item and shortcut now open the AI Search dialog directly instead of the configuration tab.
- **Statistics Page** - Added a simple statistics page showing usage overview, weekly trends chart, and monthly heatmap.

### Improvements
- **Shortcut Change** - Default "Ask" shortcut changed from Ctrl+L to Ctrl+K to avoid conflict with Calibre's built-in Edit Metadata shortcut.
- **Tutorial Update** - Added AI Search documentation to the user manual (v0.8).

### Bug Fixes
- **Fixed Syntax Error** - Resolved Python syntax error in response_panel.py that could cause plugin loading issues.
