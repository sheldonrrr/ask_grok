## GitHub Release Notes (v1.3.8)

### New Features
- **Persona Prompt** - Define your research background and goals to help AI provide more relevant responses. The persona text is prepended to all prompts.
- **Debug Logging Toggle** - New option in General Tab to enable/disable debug logging. Thanks to Comfy.n for the feedback.
- **Prompts Tab Reorganization** - Prompts configuration moved to a separate tab for better management.

### Improvements
- Reset button now properly resets all prompts and checkboxes to default values
