[b]Bug Fixes & Improvements (New in v1.3.6)[/b]

## GitHub Release Notes (v1.3.6)

- **Customizable keyboard shortcuts**: Configure the plugin shortcuts in calibre via Preferences -> Shortcuts.
- **Language follows calibre UI**: By default, the plugin uses the same language as your calibre interface (you can still override it in plugin settings).
- **Cleaner language switching**: Fixed text overlap issues when switching languages.
